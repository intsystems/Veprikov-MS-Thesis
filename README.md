Abstract
========

В работе рассматривается задача оптимального управления в системах машинного обучения с наличием скрытой петли обратной связи между предсказаниями модели и формированием обучающего набора данных. На примере рынка недвижимости показано, что если пользователи следуют рекомендациям модели, то её предсказания начинают влиять на сами данные, что может приводить к замкнутому циклу и управлению уровнем цен. Особое внимание уделяется эффекту деградации системы: некачественные рекомендации могут вызывать постепенный отток пользователей, который не отражается в стандартных метриках качества модели. Для анализа этого явления проведён вычислительный эксперимент, моделирующий поведение пользователей и динамику данных. Задача оптимального управления формализована как частично наблюдаемый марковский процесс принятия решений (PoMDP). Показано, что классические жадные алгоритмы управления могут приводить к деградации системы, несмотря на рост локальных метрик. Для предотвращения этого эффекта предложена модифицированная стратегия управления, учитывающая изменения в распределении данных. Экспериментальные результаты демонстрируют, что такой подход позволяет снизить риск деградации и поддерживать стабильность системы в долгосрочной перспективе.
